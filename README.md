# Finance-Real
Finance Real is a Progressive Web App (PWA) for personal finance management, focused on simplicity and offline-first usage. Built with pure frontend (HTML, CSS, JavaScript), it stores data locally and supports manual backup and optional Google Drive integration for controlled personal use.

Finance Real — Personal Finance App (Private Version)
A Progressive Web App (PWA) designed for personal financial management, focused on simplicity, user autonomy, and offline operation.
This version is currently used in a controlled, single-user environment to validate real-world usage before evolving into a public product.
🔹 Key Features
Income and expense tracking
Simple and efficient financial organization
Offline-first operation (localStorage)
Installable as a mobile app (PWA)
Manual data backup
Optional Google Drive integration (setup required)
Support system via PIX (copy-and-paste payment)
🔹 Architecture
Pure frontend (HTML, CSS, JavaScript)
No backend (offline-first model)
Local data persistence on user device
Structured for future scalability (cloud, authentication, monetization)
🔹 Project Status
Private testing phase:
✔️ Fully usable for daily personal use
✔️ Core features implemented
⚠️ Cloud backup requires OAuth configuration
⚠️ Not yet intended for public distribution
🔹 Goal
To validate real-world usability, data consistency, and user experience before evolving into:
multi-user version
automatic cloud sync
SaaS model
app store deployment
